# Icon 图标
语义化的矢量图形
## 何时使用
需要使用图形来对基础操作功能进行隐喻呈现，给予用户正确、友好且清晰的操作指引

## 属性
| 属性 | 类型 | 必填 | 默认值 | 说明 |
| -----|:-----:|:-----:|:-----:|----- |
| type | string | 是 | - | icon 图标的类型 |
| className | string | 否 | - | 类名 |
| size | number | 否 | - | 图标字体大小，单位px |
| style | string | 否 | - | 整体样式 |
